Get Started

Main goal of this project to show an example on how to use Akka HTTP as REST service.
This project will hep you to load your pictures by date range from your Instagram.

For more information please read README at authors Github: https://github.com/pvoznenko/akka-http-rest-example

About Akka HTTP, you can read at http://doc.akka.io/docs/akka-stream-and-http-experimental/1.0-RC4/index.html
